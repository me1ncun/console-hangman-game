<h1>Навчальний пет-проект "Шибениця".</h1>
<h2>Функціонал програми та меню консольного інтерфейсу:</h2>
<ol>
<li>При старті, програма пропонує почати нову гру або вийти з програми</li>
<li>При початку нової гри, випадково загадується слово, і гравець починає процес його відгадування</li>
<li>Після кожної введеної літери виводимо в консоль лічильник помилок, стан шибениці (намальований ASCII символами)</li>
<li>По завершенні гри виводимо результат (перемога чи поразка)</li>
</ol>
