<h2>Функционал программы и меню консольного интерфейса:</h2>
<ol>
<li>При старте программа предлагает начать новую игру или выйти из программы</li>
<li>При начале новой игры случайно загадывается слово, и игрок начинает процесс его отгадывания</li>
<li>После каждой введенной буквы выводим в консоль счетчик ошибок, состояние виселицы (нарисовано ASCII символами)</li>
<li>По завершении игры выводим результат (победа или поражение)</li>
</ol>
