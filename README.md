<h1>Навчальний пет-проект "Шибениця".</h1>
<h2>Функціонал програми та меню консольного інтерфейсу:</h2>
<li>
<ol>При старті, програма пропонує почати нову гру або вийти з програми</ol>
<ol>При початку нової гри, випадково загадується слово, і гравець починає процес його відгадування</ol>
<ol>Після кожної введеної літери виводимо в консоль лічильник помилок, стан шибениці (намальований ASCII символами)</ol>
<ol>По завершенні гри виводимо результат (перемога чи поразка) і повертаємося до стану #1 - пропозиція розпочати нову гру або вийти із програми</ol>
</li>
